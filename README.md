# Block & Focus Website

Website for the [Block & Focus](https://chrome.google.com/webstore/detail/block-focus/dcpbedhdekgkhigjgmlcbmcjoeaebbfm) Chrome extension.

Built with plain HTML, CSS, and a small vanilla JS file. No framework, no build step.

## Development

```bash
npm run dev      # start local server at http://localhost:3000
```

## Deploy

```bash
npm run deploy   # pushes src/ to the gh-pages branch
```

Images and favicons are not tracked in this repo. They live on the `gh-pages` branch and must be present in `src/img/` and `src/favicons/` locally before deploying.
